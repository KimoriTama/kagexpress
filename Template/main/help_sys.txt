# システム設定画面ヘルプテキスト

#------------------------------------------------------------------------------
# 共通

page0
	画面表示タブへ移動します。
page1
	ゲーム進行１タブへ移動します。
page2
	ゲーム進行２タブへ移動します。
page3
	テキストタブへ移動します。
page4
	サウンドタブへ移動します。
page5
	ダイアログタブへ移動します。
page6
	マウスタブへ移動します。
page7
	ショートカット一覧タブへ移動します。

back
	前の画面に戻ります。
reset
	このページの設定を初期化します。	Shiftキーを押しながらボタンを選択することで、全てのページの設定を初期化します。
title
	タイトル画面に戻ります。

# 画面設定

window	fullscreen
	ウインドウモードとフルスクリーンモードの切り替えができます。

sqr_on	sqr_off
	４：３モードと１６：９モードの切り替えができます。

noeff_on	noeff_off
	画面効果のＯＮ・ＯＦＦを設定します。

bganim_on	bganim_off
	負荷の高いアニメーション効果のＯＮ・ＯＦＦを設定します。	ＯＦＦの場合は代替の静止画が使用されます。

#showdate_on	showdate_off
#	ゲーム中のチャプター表示のＯＮ・ＯＦＦを設定します。
showdate
	ゲーム中のチャプター表示を_showDate${true?"ＯＦＦ":"ＯＮ"}に設定します。

#asicon_on	asicon_off
#	ゲーム中の進行状態アイコンの表示のＯＮ・ＯＦＦを設定します。
asicon
	ゲーム中の進行状態アイコンの表示を_asIcon${true?"ＯＦＦ":"ＯＮ"}に設定します。

showprog
	ゲーム中のボイス再生やオート待ち時間の進行メーター表示を_showProg${true?"ＯＦＦ":"ＯＮ"}に設定します。

showvreplay
	ゲーム中の音声リピートボタンの表示を_showVReplay${true?"ＯＦＦ":"ＯＮ"}に設定します。_showVReplay${true?"":"\nＯＦＦ状態でもF9のショートカットキーからリピート再生が可能です。"}

showvsave
	ゲーム中のお気に入りボイス登録ボタンの表示を_showVSave${true?"ＯＦＦ":"ＯＮ"}に設定します。_showVSave${true?"":"\nＯＦＦ状態でもバックログからお気に入りボイスを登録可能です。"}


topmost_on	topmost_off
	ＯＮにするとゲーム画面を常に手前に表示します。

esc_cancel
	ＥＳＣキーを押したときの機能を選択します。	キャンセル：右クリック相当のキャンセル動作が割り当てられます。
esc_mclick
	ＥＳＣキーを押したときの機能を選択します。	ホイールクリック：マウスタブでホイールクリックに設定した動作が割り当てられます。
esc_boss
	ＥＳＣキーを押したときの機能を選択します。	スクランブル機能：下の欄で選択されたスクランブル機能が割り当てられます。

boss_min
	スクランブル機能を選択します。	最小化：ゲームをタスクバーに収納して非表示にします。
boss_img1
	スクランブル機能を選択します。	画像１：ダミー画像１を表示します。再度ESCで解除できます。
boss_img2
	スクランブル機能を選択します。	画像２：ダミー画像２を表示します。再度ESCで解除できます。
boss_user
	スクランブル機能を選択します。	ユーザ設定：指定の画像を表示します。ボタン選択後、画像ファイルを選んでください。

curmove_on	curmove_off
	確認時のマウスカーソルの自動移動のＯＮ・ＯＦＦを設定します。

msmv_yes
	確認時のマウスカーソルの自動移動先を常に「はい」にします。
msmv_no
	確認時のマウスカーソルの自動移動先を「いいえ」にします。	ただしクイックセーブロード等の一部の確認においては「はい」になります。


curhide_none
	マウスカーソルの自動消去を設定します。
curhide_5	curhide_10	curhide_20
	マウスカーソルの自動消去を設定します。	マウスを停止した状態で指定時間放置するとカーソル表示が消去されます。

mushide_none	mushide_0
	ゲーム中に再生される音楽の曲名表示を設定します。
mushide_3	mushide_5	mushide_10
	ゲーム中に再生される音楽の曲名表示を設定します。	表示は指定時間後に自動的に消去されます。

facemode_on
	ＯＮにすると立ち絵表示中も、メッセージ窓にキャラの表情を表示させます。	（常に表示させるわけではありません。演出上、表示されない箇所も存在します）
facemode_off
	ＯＦＦにすると立ち絵表示中は、メッセージ窓にキャラの表情を表示させません。	（常に非表示となるわけではありません。演出上、表示される箇所も存在します）

touchmode_on	touchmode_off
	タッチパネル向けのUI機能のＯＮ・ＯＦＦを設定します。


#------------------------------------------------------------------------------
# ゲーム進行

gamespeed	gamespeed_rail
	ゲームの再生速度を設定します。

voicespeed	voicespeed_rail
	音声の再生速度を設定します。	速度を上げると音質が劣化しますのでご注意ください。

vspeedsync
	音声再生速度をゲーム再生速度と連動させます。	ただしx1.0〜x3.0の範囲でのみ連動する動作となります。

skipspeed	skipspeed_rail
	スキップモード中の速度を設定します。	Ctrlキーによるスキップは、この設定に関係なく常に最速になります。

hsel_on	hsel_off	hsel_out	hsel_in	hsel_m	hsel_f
	Ｈシーンの選択肢を固定するかどうかを設定します。

skipst_n
	スキップスタイルを設定します。	ノーマル：スキップ中でも全ての画面更新を表示します。
skipst_f
	スキップスタイルを設定します。	ファスト：スキップ中は立ち絵の表情やポーズ変更を省略します。
skipst_t
	スキップスタイルを設定します。	テキスト：スキップ中はテキストのみ表示し、画面や音楽は更新しません。

drama_on
	ドラマチックモードのＯＮ・ＯＦＦを設定します。	ＯＮ：オートモード中、音声があるシーンはテキストウインドウを非表示にします。
drama_off
	ドラマチックモードのＯＮ・ＯＦＦを設定します。	ＯＦＦ：オートモード中、音声があるシーンでもテキストウインドウを表示します。

use_speech
	外部の音声ツールを連動し、ボイスがないテキストを読み上げます。	メニューバーの「文字表示／テキスト読み上げ」から詳細設定が可能です。

icpreview_on	icpreview_off
	タスクバーのプレビューの表示を設定します。	この機能はWindowsVista以降で有効です。

stopdeact_on
	ゲームが非アクティブになった時の動作を設定します。	停止：ゲームの進行を一時停止して音声をミュートします。
stopdeact_off
	ゲームが非アクティブになった時の動作を設定します。	継続：ゲームの進行を停止せずそのまま継続します。


voicecut_on
	ボイスカットのＯＮ・ＯＦＦを設定します。	ＯＮ：音声再生中に次の文章に進んだ時、音声を停止します。
voicecut_off
	ボイスカットのＯＮ・ＯＦＦを設定します。	ＯＦＦ：音声再生中に次の文章に進んだ時、音声を継続します。


sysse
	システム効果音の再生を_sysseEnable${true?"無効":"有効"}にします。	効果音の音量はサウンドタブで変更できます。

sysvoice
	システムボイスの再生を_sysvoEnable${true?"無効":"有効"}にします。

sysvovolume
	システムボイスの音量を調整します。

sysvovolume_test
	システムボイスの音声サンプルを再生します。
